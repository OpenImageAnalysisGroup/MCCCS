Script and Program Files

The system is based on its main Bash-Scripts and related (Java) support programs. It uses the OME Bio-Formats, ImageJ, IAP and Weka as its main program libraries. These required libraries are downloaded automatically into the MCCCS lib folder with the corresponding MCCCS script command which is bundeled in the release zip.

To get an impression how the system works please download the latest MCCCS release. We bundled the system with several application examples to show the main functionalities. It is recommended to download and start the application examples, for further information please view the tabs installation and examples.

User documentation

Please see the online documentation for a comprehensive overview including tutorials and command descriptions. Also a short guide is available as pdf user documentation. To get an quick overview about the functionalities it is recommended to download and start the provided aplication examples.
